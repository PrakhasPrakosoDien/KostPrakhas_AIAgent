# 🏠 AI Agent KostPrakhas

AI Agent KostPrakhas adalah chatbot Customer Service berbasis Large Language Model (LLM) menggunakan Google Gemini API. Chatbot ini membantu calon penghuni mendapatkan informasi mengenai kos secara cepat, akurat, dan interaktif.

## 🚀 Features

- 🤖 AI Chat menggunakan Google Gemini
- 🏠 Informasi tipe kamar (Single Bed & Double Bed)
- 💰 Informasi harga kos
- 📶 Informasi fasilitas
- 🔒 Informasi keamanan
- 📍 Informasi lokasi kos
- 💬 Antarmuka chatbot menggunakan Streamlit

## 🛠 Tech Stack

- Python
- Streamlit
- Google Gemini API
- LangChain (opsional)
- JSON Knowledge Base

## 🎯 Use Case

Customer Service Bot untuk membantu calon penyewa mendapatkan informasi mengenai:

- Harga kamar
- Jenis kamar
- Fasilitas
- Keamanan
- Lokasi
- Peraturan kos
- Kontak
- Ketersediaan Kamar
